```bash
cd Environments/dev

terraform init
terraform validate
terraform plan
terraform apply
```

Build Azure infrastructure using reusable Terraform modules while following a clean, production-like project structure that can be extended for multiple environments such as **Dev**, **Test**, and **Production**.
